    Adicionando símbolos especiais

    Aqui temos alguns símbolos que podemos fazer atravez do HTML5:


  • Marca registrada ⇒ ®
  • Copyright ⇒ ©
  • Seta para direita ⇒
  • Seta para esquerda ⇐
  • Marca registrada em inglês (Trade mark) ⇒ ™;
  • Euro ⇒ €
  • Pound ou Libra ⇒ £
  • Centimo ⇒ ¢
  • Moeda Japonesa ⇒ ¥;
  • DElta minúsculo ⇒ δ
  • Delta maiúsculo ⇒ Δ
  • Seta para Cima duplo⇒ ⇑
  • Seta para Cima unico⇒ ↑
  • Seta para Baixo duplo⇐ ⇓
  • Seta para Baixo unico ⇐ ↓
